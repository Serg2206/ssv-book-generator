
# ssv-book-generator

**Автоматизированный инструмент для генерации профессиональных книг** под брендом **"SSVproff"**.

Генерирует:
- **SEO-оптимизированные заголовки и описания** книг с помощью ИИ.
- **Текстовое содержимое** (главы, введение, заключение) в различных жанрах (научные, художественные, хирургические, онкологические).
- **Обложки и иллюстрации** в фирменном стиле SSVproff (через API генерации изображений).
- **Форматированные книги** в различных форматах (PDF, EPUB).
- **README.md** и артефакты для архивации и поддержки open-science workflow.

Разработан профессором С.В. Сушковым для медицинского и научного сообщества.

## Установка

1.  Убедитесь, что у вас установлен **Python 3.10+** (https://python.org).
2.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/Serg2206/ssv-book-generator.git
    cd ssv-book-generator
    ```
3.  *(Рекомендуется)* Создайте виртуальное окружение:
    ```bash
    python -m venv venv
    source venv/bin/activate  # Linux/Mac
    # или
    venv\Scripts\activate     # Windows
    ```
4.  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```
5.  Скопируйте `.env.example` в `.env` и укажите свои API-ключи:
    ```bash
    cp .env.example .env
    ```
    Отредектируйте `.env`, добавив свои ключи (например, `OPENAI_API_KEY`, `SD_API_KEY`).
6.  Настройте `book_config.yaml` под свои нужды (тип книги, формат, ИИ-модель, стиль и т.д.).

## Использование

1.  Поместите файл с исходными данными (например, транскрипцией лекции, заметками, идеей) в папку `input/`.
2.  Убедитесь, что `book_config.yaml` настроен.
3.  Запустите основной скрипт:
    ```bash
    python main.py --input_file "path/to/your/input_file.txt"
    ```
4.  Результаты (финальная книга и пакет) будут сохранены в папке `output/`.
